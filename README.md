# Meta Touchscreen

### Getting Started

screen: 2160x3840

#### Run project

```
cp env.example.ts env.ts
npm install
npm start
```

#### Run project

```
cp env.example.ts env.ts
npm install
npm run build

# Install PM2 globally to manage the server process
npm add -g pm2
```

### Notes

**Convert Models**

```shell
npx gltf-pipeline -i in.gltf -b -o out.glb
```
